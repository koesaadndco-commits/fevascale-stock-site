# Heroes Worker サイト

溶接・金属加工会社「Heroes Worker（ヒーローズワーカー）」のホームページです。
※ ルートにあるフェバスケール棚卸アプリ（`../index.html`）とは独立した、別個のサイトです。

## ファイル構成

| ファイル | 内容 |
|----------|------|
| `index.html` | トップページ（このサイトの入口） |
| `works.html` | 施工事例ページ |
| `logo.png` | ロゴ画像（背景透過済み） |

ブラウザで `index.html` を開くと表示されます。
